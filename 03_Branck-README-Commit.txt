BRANCH = São Versões 
    Versão principal é a MASTER


COMMIT = Quando realiza alterações, criando arquivos, modificando etc...

README = Informações sobre o sistema


===============COMANDOS

Varre a pasta e verifica todas alterações realizadas:
    git status 

Adiciona arquivos no monitoramento do git:
    git add README.md [um arquivo por vez]
    git add -A [todos arquivos de uma vez]

Commit:
    git commit -m "Mensagem a ser inserida"

Visualizar todos commit:
    git log



Observação: O arquivo só vai para o git após o commit
